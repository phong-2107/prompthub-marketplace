Về dự án

Đây là một dự án Django chuyên nghiệp được xây dựng với cấu trúc chuẩn, sử dụng PostgreSQL làm cơ sở dữ liệu và Bootstrap 5 cho giao diện người dùng.

Tính năng chính

  • Cấu trúc dự án modular, dễ bảo trì
  • Custom User model với các trường mở rộng
  • REST API với Django REST Framework
  • Celery để xử lý tác vụ bất đồng bộ
  • Redis cho caching và message broker
  • Docker cho containerization
  • Cấu hình riêng cho development và production
